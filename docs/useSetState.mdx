This hook is like `this.setState` method of class component. It requires an object as the initial state, and can update part of the state. The new state will be merged with the old state.

## API

```ts
function useSetState<S extends object>(
  initialState: S | (() => S)
): readonly [
  state: S,
  setState: (patch: Partial<S> | ((prevState: S) => Partial<S>)) => void
];
```
