It returns the same function no matter whether the `callback` parameter is changed or not. It is used in the same way as the `callback`.

## API

```ts
function usePersist<T extends (...args: any[]) => any>(callback: T): T;
```

## Example

```tsx
import React from "react";
import { usePersist } from "@lilib/hooks";

function Example() {
  const callback = usePersist(() => {
    // Do something.
  });
  ...
}
```
