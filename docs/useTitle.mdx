Set document title.

## API

```ts
function useTitle(title: string | null, options?: { restore?: boolean }): void;
```

Params:

- **`title`:** Required document title. If it is `null`, do nothing.
- **`options`:** Options object.
  - **`options.restore`:** Restore old title on the component unmount.
