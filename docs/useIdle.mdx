Invoke the callback when the browser is idle. This hook uses [window.requestIdleCallback](https://developer.mozilla.org/en-US/docs/Web/API/Window/requestIdleCallback) by default. If the window.requestIdleCallback is non-existent, it uses setTimeout.

## API

```ts
function useIdle(
  callback: () => void
): readonly [start: () => void, cancel: () => void];
```

Params:

- **`callback`:** A function that will be invoked.

Results:

- **`start`:** Start function.
- **`cancel`:** Cancel function.

## Example

```tsx
import React from "react";
import { useIdle } from "@lilib/hooks";

function Example() {
  const [start, cancel] = useIdle(() => {
    // Do something
  });
}
```
