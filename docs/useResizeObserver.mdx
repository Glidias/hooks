Observe a element size changing.

## API

```ts
function useResizeObserver(
  target: EffectTarget<Element>,
  listener: (entries: ResizeObserverEntry[]) => void
): void;

type Nullable<T> = T | undefined | null;
type EffectTarget<T> =
  | Nullable<T>
  | (() => Nullable<T>)
  | { current: Nullable<T> };
```
