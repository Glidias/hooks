Loop run callback.

## API

```ts
function useInterval(
  callback: () => void,
  interval?: number
): readonly [start: () => void, cancel: () => void];
```

Params:

- **`callback`:** The function that is called at intervals.
- **`interval`:** Optional interval time, in milliseconds.

Results:

- **`start`:** Start loop. Automatically clear the previous loop.
- **`cancel`:** Cancel loop.
