Set up the event listener to the target.

## API

```ts
type Target<T> = T | undefined | null;
type TargetCreator<T> =
  | Target<T>
  | (() => Target<T>)
  | React.RefObject<Target<T>>
  | React.MutableRefObject<Target<T>>;

function useEventListener<
  T extends EventTarget = EventTarget,
  E extends Event = Event
>(
  target: TargetCreator<T>,
  eventName: string,
  listener: (event: E) => void,
  options?: boolean | { capture?: boolean; once?: boolean; passive?: boolean }
): void;
```

Params:

- **`target`:** Event target. It can be `undefined`, `null`, event target, a ref object or a function that returns a event target.
- **`eventName`:** Event name.
- **`listener`:** Listener function.
- **`options`:** Similar to the third parameter of the [`addEventListener`](https://developer.mozilla.org/en-US/docs/Web/API/EventTarget/addEventListener) function.
