Ensure the callback is invoked after the current paiting.

## API

```ts
function useNextFrame(
  callback: () => void
): [start: () => void, cancel: () => void];
```

Params:

- **`callback`:** The function that will be invoked.

Results:

- **`start`:** Start function.
- **`cancel`:** Cancel function.

## Example

```tsx
import React from "react";
import { useNextFrame } from "@lilib/hooks";

function Example() {
  const [start, cancel] = useNextFrame(() => {
    // Do something
  });
}
```
