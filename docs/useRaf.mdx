Use [requestAnimationFrame](https://developer.mozilla.org/en-US/docs/Web/API/window/requestAnimationFrame) to invoke the callback before the browser repainting.

## Usage

```ts
function useRaf(callback: () => void): [start: () => void, cancel: () => void];
```

### Params

- **`callback`:** the function that will be invoked.

### Results

- **`start`:** start function.
- **`cancel`:** cancel function.

## Example

```tsx
import React from "react";
import { useRaf } from "@lilib/hooks";

function Example() {
  const [start, cancel] = useRaf(() => {
    // Do something
  });
}
```
