Set document favicon.

## API

```ts
function useFavicon(
  href?: string,
  options?: { mime?: string; restore?: boolean }
): void;
```

Params:

- **`href`:** Favicon url. If it's falsy, do nothing.
- **`options.mime`:** Specify the MIME type. Get automatically according to `href` extname.
- **`options.restore`:** Restore previous favicon on the component unmount.

## Example

```tsx
import React from "react";
import { useFavicon } from "@lilib/hooks";

function Example() {
  useFavicon("https://example.com/favicon.ico", { restore: true });
  ...
}
```
