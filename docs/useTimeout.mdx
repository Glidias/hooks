Delay running callback.

## API

```ts
function useTimeout(
  callback: () => void,
  delay?: number
): [start: () => void, cancel: () => void];
```

### Params

- **`callback`:** the function that will be delayed calling.
- **`delay`:** optional delay time, in milliseconds.

### Results

- **`start`:** set a timer, automatically clear the previous timer.
- **`cancel`:** cancel timer.
