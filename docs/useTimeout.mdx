Delay running callback.

## API

```ts
function useTimeout(
  callback: () => void,
  timeout?: number
): [start: () => void, cancel: () => void];
```

### Params

- **`callback`:** A function to be executed after `timeout` time.
- **`timeout`:** Optional delay time, in milliseconds.

### Results

- **`start`:** Set a timer. Automatically clear the previous timer.
- **`cancel`:** Cancel timer.
