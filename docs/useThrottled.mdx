Throttle a value.

## API

```ts
function useThrottled<T = any>(
  value: T,
  options: number | { wait: number; leading?: boolean; trailing?: boolean }
): [
  throttledValue: T,
  {
    flush: (finalValue?: T) => void;
    cancel: () => void;
  }
];
```

Params:

- **`value`:** the value that will be changed.
- **`options`:** throttle wait time or an options object.
  - **`options.wait`:** throttle wait time, in milliseconds.
  - **`options.leading`:** change throttledValue on the leading edge, default is false.
  - **`options.trailing`:** change throttledValue on the trailing edge, default is true.

Results:

- **`throttledValue`:** throttled value.
- **`flush`:** flush funciton. The finalValue will be returned as throttledValue if it is specified.
- **`cancel`:** cancel throttle timer.
