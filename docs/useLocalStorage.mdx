Get or set localStorage.

## API

```ts
function useLocalStorage<T = any>(
  key: string,
  defaultValue?: T,
  options?: {
    serialize?: (value: T) => string;
    deserialize?: (raw: string) => T;
  }
): [
  value: T | null,
  actions: {
    set: (newValue: T) => void;
    remove: () => void;
  }
];
```

Params:

- **`key`:** `localStorage` key.
- **`defaultValue`:** Default value.
- **`options.serialize`:** Options object.
  - **`options.serialize`:** Serialization function, default is `JSON.stringify`.
  - **`options.deserialize`:** Deserialization function, default is `JSON.parse`.

Results:

- **`value`:** Parsed by the deserialization function.
- **`set`:** The function that used to set localStorage.
- **`remove`:** The function that used to remove localStorage.
