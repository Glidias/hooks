Debounce a value.

## API

```ts
function useDebounced<T = any>(
  value: T,
  options: number | { wait: number; leading?: boolean; trailing?: boolean }
): [
  debouncedValue: T,
  {
    flush: (finalValue?: T) => void;
    cancel: () => void;
  }
];
```

Params:

- **`value`:** the value that will be changed.
- **`options`:** debounce wait time or an options object.
  - **`options.wait`:** debounce wait time, in milliseconds.
  - **`options.leading`:** change debouncedValue on the leading edge, default is false.
  - **`options.trailing`:** change debouncedValue on the trailing edge, default is true.

Results:

- **`debouncedValue`:** debounced value.
- **`flush`:** flush funciton. The finalValue will be returned as debouncedValue if it is specified.
- **`cancel`:** cancel debounce timer.
