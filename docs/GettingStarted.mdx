[`@lilib/hooks`](https://www.npmjs.com/package/@lilib/hooks) is a React hook library.

## Install

```shell
npm install --save @lilib/hooks
```

## Example

```tsx
import React from "react";
import { useMount } from "@lilib/hooks";

function Example() {
  useMount(() => {
    console.log("Component is mounted");
  });
  ...
}
```
